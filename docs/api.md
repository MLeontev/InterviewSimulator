# Interview Simulator API

Спецификация REST API онлайн-тренажера технических собеседований.

Версия API: `v1`

Базовый префикс: `/api/v1`

Формат данных: `application/json`

## Авторизация

Защищенные эндпоинты используют OAuth 2.0 Authorization Code Flow + PKCE через Keycloak.

Для вызова защищенных эндпоинтов необходимо передавать access token в заголовке:

```http
Authorization: Bearer <access_token>
```

Эндпоинты регистрации и получения пресетов доступны без авторизации. Остальные эндпоинты требуют авторизации кандидата.

## Формат ошибок

Стандартная ошибка API:

| Поле          | Тип      | Описание                                 |
|---------------|----------|------------------------------------------|
| `code`        | `string` | Машиночитаемый код ошибки                |
| `description` | `string` | Человекочитаемое описание причины ошибки |

Ошибка валидации:

| Поле          | Тип                        | Описание                                 |
|---------------|----------------------------|------------------------------------------|
| `code`        | `string`                   | Машиночитаемый код ошибки                |
| `description` | `string`                   | Человекочитаемое описание причины ошибки |
| `errors`      | `object<string, string[]>` | Ошибки валидации по полям запроса        |

Часто используемые статусы:

| Код   | Значение                                                                                   |
|-------|--------------------------------------------------------------------------------------------|
| `400` | Некорректное тело запроса или ошибка валидации входных данных                              |
| `401` | Пользователь не авторизован                                                                |
| `403` | Доступ к ресурсу запрещен                                                                  |
| `404` | Запрошенный ресурс не найден                                                               |
| `409` | Конфликт состояния ресурса                                                                 |
| `422` | Запрос корректен синтаксически, но не может быть выполнен из-за текущего состояния системы |
| `502` | Ошибка взаимодействия с внутренним модулем                                                 |

## Пользователи

### Зарегистрировать пользователя

```http
POST /api/v1/users
```

Email должен быть уникальным. После регистрации пользователь создается в сервисе аутентификации и в базе приложения.

Тело запроса:

| Поле       | Тип      | Описание                             |
|------------|----------|--------------------------------------|
| `email`    | `string` | Адрес электронной почты пользователя |
| `password` | `string` | Пароль пользователя                  |

Ответ `201 Created`:

| Поле         | Тип      | Описание                                            |
|--------------|----------|-----------------------------------------------------|
| `userId`     | `uuid`   | Идентификатор пользователя в базе приложения        |
| `identityId` | `string` | Идентификатор пользователя в сервисе аутентификации |

Ошибки:

| HTTP  | `code`                         | `description`                                      | Описание                                                                              |
|-------|--------------------------------|----------------------------------------------------|---------------------------------------------------------------------------------------|
| `400` | `VALIDATION_ERROR`             | Возникли одна или несколько ошибок валидации       | Детали ошибок передаются в поле `errors`                                              |
| `409` | `IDENTITY_EMAIL_IS_NOT_UNIQUE` | Пользователь с указанным email уже зарегистрирован | Необходимо использовать другой email или выполнить вход в существующую учетную запись |

### Получить текущего пользователя

```http
GET /api/v1/users/me
```

Требуется авторизация.

Ответ `200 OK`:

| Поле         | Тип      | Описание                                            |
|--------------|----------|-----------------------------------------------------|
| `id`         | `uuid`   | Идентификатор пользователя в базе приложения        |
| `email`      | `string` | Адрес электронной почты пользователя                |
| `identityId` | `string` | Идентификатор пользователя в сервисе аутентификации |

Ошибки:

| HTTP  | `code`           | `description`                                      | Описание                                                                  |
|-------|------------------|----------------------------------------------------|---------------------------------------------------------------------------|
| `401` | -                | -                                                  | Пользователь не авторизован: токен доступа отсутствует или недействителен |
| `404` | `USER_NOT_FOUND` | Пользователь не найден. Сначала зарегистрируйтесь. | Пользователь авторизован в Keycloak, но не найден в базе приложения       |


| Поле   | Тип      | Описание                            |

| `id`   | `uuid`   | Идентификатор пресета собеседования |
| `name` | `string` | Название пресета собеседования      |

### Получить пресет собеседования

```http
GET /api/v1/interview-presets/{id}
```

Параметры пути:

| Параметр | Тип    | Описание                            |
|----------|--------|-------------------------------------|
| `id`     | `uuid` | Идентификатор пресета собеседования |

| Поле             | Тип               | Описание                                             |

| `id`             | `uuid`            | Идентификатор пресета собеседования                  |

## Сессии собеседований

### Создать сессию собеседования

```http
POST /api/v1/interview-sessions
```

Требуется авторизация.

У кандидата может быть только одна активная сессия одновременно. Сессия действительна 1 час с момента создания.

Тело запроса:

| Поле                | Тип    | Описание                            |
|---------------------|--------|-------------------------------------|
| `interviewPresetId` | `uuid` | Идентификатор пресета собеседования |

Успешный ответ: `201 Created`, тело ответа отсутствует.

Ошибки:

| HTTP  | `code`                  | `description`                                      | Описание                                                                       |
|-------|-------------------------|----------------------------------------------------|--------------------------------------------------------------------------------|
| `400` | `VALIDATION_ERROR`      | Возникли одна или несколько ошибок валидации       | Некорректное тело запроса или отсутствует обязательный `interviewPresetId`     |
| `401` | -                       | -                                                  | Пользователь не авторизован: токен доступа отсутствует или недействителен      |
| `403` | -                       | -                                                  | Доступ к ресурсу запрещен: пользователь не имеет доступа к выполнению операции |
| `404` | `USER_NOT_FOUND`        | Пользователь не найден. Сначала зарегистрируйтесь. | Пользователь авторизован в Keycloak, но не найден в базе приложения            |
| `404` | `PRESET_NOT_FOUND`      | Пресет интервью не найден                          | Выбранный пресет отсутствует                                                   |
| `409` | `ACTIVE_SESSION_EXISTS` | У кандидата уже есть активная сессия интервью      | Перед созданием новой сессии необходимо завершить текущую                      |
| `422` | `NO_QUESTIONS`          | Не удалось получить вопросы для сессии интервью    | Для выбранного пресета не найден подходящий набор заданий                      |
| `502` | `QUESTION_BANK_ERROR`   | Текст ошибки модуля банка вопросов                 | Ошибка при обращении к модулю банка вопросов                                   |

```http

```

Требуется авторизация.

Возвращаются только завершенные или оцененные сессии текущего кандидата. Активная сессия не включается в историю.

Ответ `200 OK`: массив объектов `InterviewSessionHistoryItem`.

| Поле                  | Тип               | Описание                                                                    |
|-----------------------|-------------------|-----------------------------------------------------------------------------|
| `sessionId`           | `uuid`            | Идентификатор сессии собеседования                                          |
| `interviewPresetId`   | `uuid`            | Идентификатор пресета собеседования                                         |
| `interviewPresetName` | `string`          | Название пресета собеседования                                              |
| `status`              | `InterviewStatus` | Статус сессии                                                               |
| `sessionVerdict`      | `SessionVerdict`  | Итоговый вердикт сессии                                                     |
| `startedAt`           | `date-time`       | Дата и время начала сессии                                                  |
| `plannedEndAt`        | `date-time`       | Плановая дата и время завершения сессии                                     |
| `finishedAt`          | `date-time?`      | Фактическая дата и время завершения сессии                                  |
| `totalQuestions`      | `int`             | Общее количество заданий в сессии                                           |
| `completedQuestions`  | `int`             | Количество заданий, выполнение которых было начато, завершено или пропущено |

Ошибки:

| HTTP  | `code`           | `description`                                      | Описание                                                                     |
|-------|------------------|----------------------------------------------------|------------------------------------------------------------------------------|
| `401` | -                | -                                                  | Пользователь не авторизован: токен доступа отсутствует или недействителен    |
| `403` | -                | -                                                  | Доступ к ресурсу запрещен: пользователь не имеет доступа к просмотру истории |
| `404` | `USER_NOT_FOUND` | Пользователь не найден. Сначала зарегистрируйтесь. | Пользователь авторизован в Keycloak, но не найден в базе приложения          |

```http

```

Требуется авторизация.

Повторную ИИ-оценку можно запустить только для сессии, у которой предыдущая ИИ-оценка завершилась ошибкой.

Параметры пути:

| Параметр    | Тип    | Описание                           |
|-------------|--------|------------------------------------|
| `sessionId` | `uuid` | Идентификатор сессии собеседования |

Ошибки:

| HTTP  | `code`                   | `description`                                      | Описание                                                                                         |
|-------|--------------------------|----------------------------------------------------|--------------------------------------------------------------------------------------------------|
| `401` | -                        | -                                                  | Пользователь не авторизован: токен доступа отсутствует или недействителен                        |
| `403` | -                        | -                                                  | Доступ к ресурсу запрещен: кандидат пытается повторно оценить сессию, к которой не имеет доступа |
| `404` | `USER_NOT_FOUND`         | Пользователь не найден. Сначала зарегистрируйтесь. | Пользователь авторизован в Keycloak, но не найден в базе приложения                              |
| `404` | `SESSION_NOT_FOUND`      | Сессия не найдена                                  | Сессия отсутствует или не принадлежит текущему кандидату                                         |
| `422` | `SESSION_NOT_FAILED`     | ИИ-оценка сессии не была завершена с ошибкой       | Повтор доступен только после ошибки итоговой ИИ-оценки                                           |
| `422` | `NO_FAILED_AI_QUESTIONS` | Нет заданий с ошибкой ИИ-оценки                    | В сессии нет заданий, требующих повторной ИИ-оценки                                              |

### Изменить состояние текущей сессии

```http
PATCH /api/v1/interview-sessions/current
```

Требуется авторизация.

Поддерживается только статус `Finished`. Вызов завершает текущую активную сессию досрочно.

Тело запроса:

| Поле     | Тип                           | Описание                       |
|----------|-------------------------------|--------------------------------|
| `status` | `InterviewSessionStatusPatch` | Новое состояние текущей сессии |

Возможные значения `status`:

| Значение   | Описание                 |
|------------|--------------------------|
| `Finished` | Завершить текущую сессию |

Успешный ответ: `204 No Content`, тело ответа отсутствует.

Ошибки:

| HTTP  | `code`                       | `description`                                      | Описание                                                                             |
|-------|------------------------------|----------------------------------------------------|--------------------------------------------------------------------------------------|
| `400` | `VALIDATION_ERROR`           | Возникли одна или несколько ошибок валидации       | Некорректное тело запроса или неподдерживаемое значение `status`                     |
| `401` | -                            | -                                                  | Пользователь не авторизован: токен доступа отсутствует или недействителен            |
| `403` | -                            | -                                                  | Доступ к ресурсу запрещен: пользователь не имеет доступа к завершению текущей сессии |
| `404` | `USER_NOT_FOUND`             | Пользователь не найден. Сначала зарегистрируйтесь. | Пользователь авторизован в Keycloak, но не найден в базе приложения                  |
| `404` | `SESSION_NOT_FOUND`          | Текущая сессия интервью не найдена                 | У кандидата нет активной сессии                                                      |
| `422` | `SESSION_CANNOT_BE_FINISHED` | Сессию нельзя завершить в текущем статусе          | Завершить можно только сессию в статусе `InProgress`                                 |

### Изменить состояние текущего задания

```http
PATCH /api/v1/interview-sessions/current/question
```

Требуется авторизация.

Поддерживаются только статусы `InProgress` и `Skipped`. `InProgress` запускает выполнение задания. `Skipped` пропускает
текущее задание.

Тело запроса:

| Поле     | Тип                            | Описание                         |
|----------|--------------------------------|----------------------------------|
| `status` | `InterviewQuestionStatusPatch` | Новое состояние текущего задания |

Возможные значения `status`:

| Значение     | Описание                   |
|--------------|----------------------------|
| `InProgress` | Начать выполнение задания  |
| `Skipped`    | Пропустить текущее задание |

Успешный ответ: `204 No Content`, тело ответа отсутствует.

Ошибки:
